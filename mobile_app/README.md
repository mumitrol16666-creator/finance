# Finance Tracker Mobile App (Flutter)

This directory contains the cross-platform mobile application for the Finance Tracker, built using Flutter.

## Setup Instructions

Once the Flutter SDK download is complete:
1. Extract the Flutter ZIP file to your preferred location (e.g., `C:\src\flutter`).
2. Add the `flutter/bin` directory to your user PATH environment variable.
3. Run `flutter doctor` in your terminal to verify dependencies.
4. Open this `mobile_app` folder and initialize a new Flutter project.
